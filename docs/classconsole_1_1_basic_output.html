Detailed Description

template<class CharT = char, class Traits = std::char_traits<CharT>>
class console::BasicOutput< CharT, Traits >

灵活的输出控制类，支持链式调用和多种参数。

Template Parameters
CharT字符类型。
Traits字符特征类型。

该类允许用户指定分隔符、结尾符和是否立即刷新，并提供 operator() 重载 来接收任意数量的参数进行格式化输出。 默认全局实例 Output 的行为类似于 Python 的 print 函数。

使用示例：

print("Hello", "world"); // 输出 "Hello world\n" 并刷新
print(); // 只输出换行
BasicOutput err(std::cerr, "", "\n", true);
err("Error:", code); // 输出到 stderr，无分隔符
