Detailed Description

template<class CharT = char, class Traits = std::char_traits<CharT>>
class console::BasicLogging< CharT, Traits >

日志记录器，支持多级别、颜色输出和自动时间戳。

Template Parameters
CharT字符类型。
Traits字符特征类型。

使用 BasicOutput 类进行格式化输出，可设置最低日志级别或单独控制每个级别的开关。 FATAL 级别会抛出 FatalLogging 异常。
