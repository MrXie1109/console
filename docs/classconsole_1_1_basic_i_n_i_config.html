Detailed Description

template<class CharT = char>
class console::BasicINIConfig< CharT >

INI 配置文件操作类。

Template Parameters
CharT字符类型，支持 char 和 wchar_t。

支持加载、保存、读取、修改 INI 格式的配置文件，本类的 INI 变种支持空节头和空键。
