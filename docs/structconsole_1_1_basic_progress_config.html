Detailed Description

template<class CharT = char, class Traits = std::char_traits<CharT>>
struct console::BasicProgressConfig< CharT, Traits >

进度条显示配置。

Template Parameters
CharT字符类型。
Traits字符特征类型。

该结构体控制进度条的外观和行为，包括输出流、宽度、填充字符、前后缀以及是否显示百分比。
