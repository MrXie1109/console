Detailed Description

表示以纳秒为单位的时间量，支持单位转换、算术运算和自动选择合适的输出单位。

内部使用 long long 存储纳秒数，提供 ns()、us()、ms()、s()、min()、hr() 等转换函数。 输出流会根据时间大小自动选择最合适的单位（hr/min/s/ms/μs/ns）。
