Detailed Description

简单的线程池实现，模仿了 Python 的 concurrent.futures.ThreadPoolExecutor 接口。

该模块提供了一个高效的线程池实现，支持任务提交、批量映射执行、 等待所有任务完成、优雅关闭等功能。适用于需要并发执行多个独立任务的场景。
